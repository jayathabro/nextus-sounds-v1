!play <query>              - ගීතයක් ධාවනය කරන්න / පෝලිමයට එකතු කරන්න
!p <query>                 - (alias) play command හි කෙටි ආකාරය
!pause                     - සක්දීම් ධාවනය වෙමින් තිබේහින් pause කරන්න
!resume / !unpause         - ධාවනය කිරීම දිගු කරන්න
!skip / !s / !next         - වර්තමාන ගීතය ස�kip කරන්න
!stop / !leave / !dc       - ධාවනය නික්මීම + පෝලිම හිස් කරන්න
!queue / !q / !පෝලිම     - පෝලිම දැක ගන්න (page එකක් pass කළ හැක)
!nowplaying / !np / !current - වර්තමාන play වෙමින් තිබෙන ගීතය දැක ගන්න
!volume <0-150> / !vol / !v - හඬ විෂ්කම්භය සකසන්න

Loop & Playback Controls:
--------------------------------------------------------------------------------
!loop <off|track|queue> / !repeat - Loop mode එක තෝරන්න
!shuffle                   - පෝලිම shuffle කරන්න
!seek <seconds>            - ගීතයේ තැනකට යයින්ට යන්න
!remove <pos> / !rm        - පෝලිමෙන් අයිතිම ඉවත් කරන්න
!clearqueue / !cq / !clear  - පෝලිම සම්පූර්ණයෙන් හිස් කරන්න

================================================================================
!sb <name>                 - Custom sound එකක් ධාවනය කරන්න

!sb add <name> (Admin)      - Administrator අයිතිය ඇතිව sound එකක් එකතු කරන්න
!sb remove <name> (Admin)   - Administrator අයිතිය ඇතිව sound එකක් ඉවත් කරන්න
!sb stop                   - soundboard playback නික්ම කරන්න

================================================================================
🔒 Owner Command Only (Bot Owner) 🔒
================================================================================
!shutdown / !stopbot / !off    - Bot ကို အင်္ကအင်းပြင်ဆင်ချက်အနေဖြင့် မည်သို့မဆိုင်သော်လည်းကောင်းကောင်းမွန်စွာဖြင်ပြင်ဆင်ချက်ပြုလုပ်နိုင်ပါသည်
!restart                    - Bot ကို ပြန်လည်စတင်ထုတ်လုပ်ချက်လုပ်ပါသည်
!reload <cog>               - Cog တစ်ခုကို ပြန်လည်တင်သွင်းပါသည် (ဥပမာ: !reload music)
!load <cog>                 - Cog တစ်ခုကို အားလုံးအားလုံးတင်သင့်ပါသည်
!unload <cog>               - Cog တစ်ခုကို အားလုံးအားလုံးမတင်ပါသည်
!cogs                       - တင်သင့်သော့တစ်ခုတို့ကို အားလုံးအားလုံးပြရန်လုပ်ပါသည်
!guilds / !servers          - Bot ရဲ့အပါအဝင်ရှိသမုံတက်ထားသည့်အီးမီးယာဂျာများကိုပြရန်လုပ်ပါသည်
!broadcast <message>        - အီးမီးယာဂျာအားလုံးကို DM ပေးပို့ရန်လုပ်ပါသည်
!blacklist <user> <reason>   - အသုံးပြုသူကိုကာကွယ်ထားသည်လုပ်ပါသည်
!sync                       - Slash commands ကိုအားလုံးအားလုံးတင်သင့်ပါသည်
!update                     - Git မှာ့ဆုပေးပို့ရန်နဲ့ reload လုပ်ပါသည်
